O arquivo Benford.R contém o código responsável pelo teste inicial. O Benford2.R contém o código do teste secundário. Para o teste final, basta descomentar a linha " num = unique(num) " d arquivo Benford2.R.

A única modificação necessária (além dessa citada acima) para rodar os códigos é substituir a parte onde está escrito filepath pelo caminho referente ao arquivo que se quer rodar.

Vale ressaltar que é necessário ter instalado previamente tanto R quanto as bibliotecas moments (calcula a obliquidade) e BenfordTests (possui as funções para análise estatística dos dados).
